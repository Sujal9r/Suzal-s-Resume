### Run locally
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
npm run preview
```

### Customize
- Update links in `src/components/Navbar.jsx` and `Contact.jsx`
- Replace hero image in `src/components/Hero.jsx`
- Edit projects in `src/components/Projects.jsx`
